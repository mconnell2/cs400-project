Food Meal Planner

Overview

Welcome to the CS400 Food and Meal Planner System. This helpful application takes the guesswork out of meal-planning by supporting a simple, intuitive workflow for food and meal management With this application you can manage a list of food items and combine these items into simple meals and quickly analyze the nutritional content of your meals.

Getting Started

  • Click the Load from File button and choose your food *.csv file.
  • Add a food item to the meal using the Right arrow button
  • Remove a food item from the meal using the Left arrow button
  • Have fun, and stay healthy!

    • Features

    The Food List

    The Food List is your master list of all available food items. You can populate the list from a saved food.csv file or by adding new food items manually. Loaded food items may be added to the meal using the Right and Left arrow buttons. You can also restrict the number of food items displayed using custom filters

    Loading food from a file

    Food items may be loaded from a food.csv file. Click the Load from File button to open a file browser and select the food file to load. Multiple food files may be loaded into the food list.

    Adding new food items

    New food items may be defined using the Add New Food button. This button will open up a form where you can define a new food item and specify its nutritional information.

    Filtering food

    You can restrict the items displayed on the Food List by defining custom filters using the filter button . The filter feature allows you to restrict the list to only foods whose names contain specific text.

    Nutrition Filters
    You can also define one or more nutrition filters in the following format:

    nutrient [==|<=|>=] value

    Define nutrition filters by specifying a nutrient and a comparison rule. You may specify as many nutirent rules as you like. Food items must satisfy all nutrient rules to appear in the list. Available nutrients are:

    1. calories
    2. carbohydrates
    3. protein
    4. fat
    5. fiber

    Saving Food Items

    You can save your food list for re-use in a food.csv file. Click the Save to File button to open a file browser and select/create the file to save your food items to.

    Meal List

    The Meal List tracks the individual food items that comprise your meal. A nutritional analysis of the meal is automatically updated as food items are added or removed from the list. You can add or remove items individually using the Right and Left arrow buttons. You can also clear the entire meal using the Clear Meal button.
